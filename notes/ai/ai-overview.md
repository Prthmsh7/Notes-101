## What this topic covers

AI revision gets easier when you separate model capability, prompting, evaluation, and deployment constraints.

## Revision checkpoints

- Know the difference between training, fine-tuning, and inference.
- Review prompts as system design, not magic text.
- Measure outputs with repeatable evaluation criteria.

```text
prompt -> model -> output -> review -> iterate
```
